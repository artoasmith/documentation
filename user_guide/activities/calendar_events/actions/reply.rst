Respond to a Calendar Event
^^^^^^^^^^^^^^^^^^^^^^^^^^^


.. contents:: :local:

From the Calendar Events Grid
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. begin

To respond to a calendar event when viewing a list of events:

1. In the main menu, navigate to **Activities>Calendar Events**.

2. In the grid on the **Calendar Events** page, click the ellipsis menu at the end of the row with the event you want to reply to and then click either the |IcCheck| **Accept**, |IcInviteMaybe2| **Tentative**, or |IcTimes| **Decline** icon depending on whether you are going to attend the event.

   |

   .. image:: /user_guide/img/activities/activities_events_actions_respond_fromgrid.png

   |

.. finish

From the Calendar Event View Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
To respond to a calendar event when viewing the event details:

1. In the main menu, navigate to **Activities>Calendar Events**.

2. In the grid on the **Calendar Events** page, click the required event.

   Alternatively, you can click the ellipsis menu at the end of the corresponding row and then click the |IcView| **View** icon to get more details about the event before responding.

3. On the calendar event view page, click either the :guilabel:`Accept`, :guilabel:`Tentative`, or :guilabel:`Decline` button in the upper-right corner depending on whether you are going to attend the event.

   |

   .. image:: /user_guide/img/activities/activities_events_actions_response_viewpage.png

   |
