Clone |oro_app_name| source code to */usr/share/nginx/html/oroapp* folder:

.. code:: bash

   cd /usr/share/nginx/html
   git clone -b 3.0 https://github.com/oroinc/crm-enterprise-application.git oroapp
   cd oroapp

..
   The *branch* value (*3.0* in this example) could be changed to any published
   `release tag <https://github.com/oroinc/crm-enterprise-application/releases>`_ from 3.0 branch of
   the |oro_app_name| application (for example, 3.0, 3.0.1, etc.).
